# nodemon

___

описание - nodemon
<br/>
если demon видит ,что код в файле изменился
<br/>
тогда он перезагружает сервер

___

установка

``` bash
npm i nodemon
```
теперь для запуска сервера нужно писать не node index.js
``` bash
nodemon index.js
```
